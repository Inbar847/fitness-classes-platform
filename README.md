# Fitness Classes Platform

Full-stack web platform for managing and booking fitness classes (exact MVP from attached PRDs).

**Tech Stack (mandatory)**
- Frontend: React 18 + TypeScript + Vite + Tailwind
- Backend: FastAPI + SQLAlchemy 2.0 + PostgreSQL
- Infra: Docker + docker-compose

**Quick Start**
1. `cp .env.example .env`
2. `docker compose up --build`
3. Open http://localhost:5173 (frontend) and http://localhost:8000 (backend)

Follow the strict 8-phase order. All rules from PRD, ERD, and Engineering Guidelines enforced.
